# 🐱 Kiriosity ✨

Kiriosity is a lightweight desktop app that captures on-screen text and translates it instantly.

Perfect for games, chats, or anything where you need quick translations without switching windows.

## 🚀 Features

* 📸 Select any area of your screen (F8)
* 🌍 Automatic translation to English
* ✍️ Manual translation box
* 💖 Clean pink UI + cute cat branding
* ⚡ Fast and simple to use

---

## 🖥️ How to Use

1. Launch **Kiriosity.exe**
2. Press **F8**
3. Drag to select text on your screen
4. Translation will appear automatically

### Manual Translation

* Type text in the right panel
* Select a language
* Click **Translate**
